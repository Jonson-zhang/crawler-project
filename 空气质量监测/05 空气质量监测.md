
## 一、目的

请求地址：https://www.aqistudy.cn/historydata/ ，获取各地空气质量的数据

## 二、加密参数观察

1. 网站使用无限 debugger 技术，使用我们的油猴脚本可以跳过检测
2. 网站使用缓存技术，获取数据后，写入 localstorage，然后解密显示。调试时，尽可能点击不同城市，查看数据
3. 网站真正请求数据接口的是historyapi.php
4. 请求头中没有需要关注的参数
5. 请求方式是 POST
6. 载荷有参数 ：hA4Nse2cT。它的内容是加密的，长度是 344 位。点击不同城市，发送请求时，该参数内容是变化的
7. 响应结果是加密的

## 三、逆向

根据分析，重点处理两处：
- 请求载荷中的加密参数
- 加密的响应数据

1、响应数据的解密，一般采用 hook json 拦截解密函数；而载荷中的参数，可以使用 hook json，也可以全局搜索关键字。
2、全局搜索“hA4Nse2cT”，没有找到，于是采用 hook json 进行拦截。
3、由于网站使用无限 debugger 反调试，所有启用油猴的反 debugger 脚本

```
var localStorageUtil = {
  save: function(name, value) {
    var text = JSON.stringify(value);
    text = BASE64.encrypt(text);
    text = AES.encrypt(text, aes_local_key, aes_local_iv);
    try {
      localStorage.setItem(name, text);
    } catch (oException) {
      if (oException.name === 'QuotaExceededError') {
        console.log('Local limit exceeded');
        localStorage.clear();
        localStorage.setItem(name, text);
      }
    }
  },
  check: function(name) {
    return localStorage.getItem(name);
  },
  getValue: function(name) {
    var text = localStorage.getItem(name);
    var result = null;
    if (text) {
      text = AES.decrypt(text, aes_local_key, aes_local_iv);
      text = BASE64.decrypt(text);
      result = JSON.parse(text);
    }
    return result;
  },
  remove: function(name) {
    localStorage.removeItem(name);
  }
};
```
