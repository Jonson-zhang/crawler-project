## WASM 逆向工具

两个命令行工具，npm 全局安装即可：

```bash
npm install -g wabt binaryen
```

| 工具 | 所属包 | 核心命令 | 作用 |
|------|--------|----------|------|
| **wabt** (WebAssembly Binary Toolkit) | `wabt` | `wasm-objdump -x` | 查看 import/export 清单、函数签名、段结构 |
| | | `wasm-objdump -d` | 完整反汇编，搜索 `unreachable` 定位崩溃点 |
| | | `wasm-decompile` | 反编译出类 C 伪代码，阅读算法结构 |
| | | `wasm2c` | 将 WASM 转为可编译的 C 代码 |
| **binaryen** | `binaryen` | `wasm-dis` | 二进制 → WAT 文本 |
| | | `wasm-opt` | WASM 优化/插桩/转换 |

实际效果（对 lizhi.wasm）：
- `wasm-objdump -x lizhi.wasm` → 发现 403 个内部函数、30 个 import、8 个 export
- `wasm-objdump -d lizhi.wasm | grep unreachable` → 找到 200+ 个检测埋点
- 精确 crash 定位：偏移 0x11b09，紧接 `i32.const 4736` 给 global 赋值后
- 关键发现：`M.push(void 0, null, !0, !1)` 必须用 `!0`/`!1`，不能用 `true`/`false`
